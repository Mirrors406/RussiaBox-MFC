# RussiaBox-MFC

使用MFC单文档制作的简易俄罗斯方块游戏.

Windows程序设计作业，开发环境为VisualStudio 2013.

源码中包含大量的注解，适合初学者使用.

如果没有什么思路的话，可以参考xmind思维导图，内含两张导图：1.功能详细实现、2.基本流程，因为第一张表写的比较早，我后期函数名、变量名以及部分实现有所更改，应该不太影响.

！！！！！不要直接使用本代码应付作业噢！！！！！

===== RBox_1.0 功能实现 =====

1.开始游戏

2.方块的创建

3.下一方块的预览

4.方块的自由下落

5.方块的碰撞检测（包含变形时碰撞检测）

6.键盘控制方块的移动

7.游戏结束判定

===== RBox_1.1 功能更新 =====

1.添加了游戏暂停

2.添加了按下数字键改变下一方块类型的功能
